Вариант 28. С начала суток прошло N секунд (N — целое). Найти количество
полных минут, прошедших с начала последнего часа.


Начало  ------->    Ввод N (количество секунд) ------> Вычисление минут: M = N // 60    ----->   Вывод "N секунд прошло сначала дня, что эквивалентно M минутам" ----->  Конец

Code:
N = int(inpit('Введите количество секунд прошедших сначала дня: '))#Переменная
print(f'{N} Секунд прошло сначала дня, что эквивалентно {N // 60} минутам')#Вывод
